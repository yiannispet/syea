- ✅ Responsive (λειτουργεί σε κινητά & tablets)
- ✅ Dark Mode
- ✅ Εξαγωγή/Εισαγωγή δεδομένων (JSON backup)
- ✅ Εξαγωγή σε Excel (CSV)
- ✅ Αποθήκευση στον browser (localStorage)
- ✅ Λειτουργεί offline

## 💾 Αποθήκευση Δεδομένων

Τα δεδομένα αποθηκεύονται τοπικά στον browser σας (localStorage).

⚠️ **Σημαντικό:** Χρησιμοποιήστε τη λειτουργία **Εξαγωγή** για backup!

## 📤 Backup & Restore

1. **Εξαγωγή:** Κλικ στο `📤 Εξαγωγή` - Κατεβάζει αρχείο JSON
2. **Εισαγωγή:** Κλικ στο `📥 Εισαγωγή` - Επιλέξτε το JSON αρχείο
